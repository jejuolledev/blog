## 오늘의 작업

드디어 블로그를 새로 만들기로 했다. Henry Heffernan의 포트폴리오를 보고 영감을 받아서
PC 부팅 애니메이션으로 시작하는 블로그를 만들기로 결정!

### 계획

1. Three.js로 3D 컴퓨터 모델 구현
2. 부팅 시퀀스 애니메이션
3. 모니터 안에 실제 콘텐츠 표시

### 기술 선택

- **Next.js 13**: App Router 사용
- **Three.js + React Three Fiber**: 3D 렌더링
- **Contentlayer**: MDX 콘텐츠 관리
- **Tailwind CSS**: 스타일링

오늘은 기본 구조 설계만 하고 내일부터 본격적으로 코딩 시작!
