## 프로젝트 소개

제주에서 살면서 느낀 점은 관광객들이 잘 모르는 숨겨진 명소가 정말 많다는 것입니다.
이 프로젝트는 그런 로컬 스팟들을 공유하고 발견할 수 있는 플랫폼입니다.

## 주요 기능

- **인터랙티브 지도**: MapBox GL JS를 활용한 부드러운 지도 경험
- **카테고리 필터**: 카페, 음식점, 자연, 문화 등 다양한 카테고리
- **사용자 리뷰**: 방문자들의 실제 후기 공유
- **북마크 기능**: 가고 싶은 장소 저장

## 기술 스택

- **Frontend**: React, TypeScript, Tailwind CSS
- **Map**: MapBox GL JS
- **Backend**: Supabase (PostgreSQL + Auth)
- **Deployment**: Vercel

## 배운 점

MapBox와 React를 연동하면서 성능 최적화의 중요성을 깨달았습니다.
특히 마커가 많아질 때 클러스터링 기법을 적용해 렌더링 성능을 크게 개선할 수 있었습니다.
