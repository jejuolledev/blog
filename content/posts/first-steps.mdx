## 시작하기 전에

디자인 시스템은 한 번에 완성되지 않습니다. 작은 컴포넌트부터 시작해 점진적으로 범위를 넓혀가야 합니다.

## 체크리스트

- 핵심 톤앤매너 정의
- 버튼, 입력 필드, 카드처럼 자주 쓰는 컴포넌트부터 정리
- 문서화 전략 확정

## 코드 예시

```ts
const tokens = {
  color: {
    primary: '#225546',
    surface: '#f8f6f3'
  }
};
```
